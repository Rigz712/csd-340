5 Things I've Learned Since the Course Started

  • Capital letters can be as readable as lowercase letters depending on context and usage.
  • Reading on screens is harder than on paper, and proper font size varies by device type.
  • Recognition is easier than recall for user interfaces, and successful designs use this principle.
  • Emotional memories can be inaccurate, impacting user feedback and experience design.
  • People scan web pages based on cultural reading patterns, using peripheral vision more than central vision.
